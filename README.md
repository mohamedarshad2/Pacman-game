# Pac-Man Game in Python

## Overview
This project is a Pac-Man style arcade game developed in Python using Pygame. It includes player movement, ghost movement, collectible dots, power-ups, scoring, lives, and a tile-based board layout.

## Features
- Pac-Man player movement
- Multiple ghost enemies with distinct behavior
- Dot and power pellet collection
- Score tracking
- Lives system
- Tile-based maze layout
- Sprite-based rendering using custom assets

## Technologies Used
- Python
- Pygame
